# Python4k Hub

Персональная стартовая страница для быстрого доступа ко всем self-hosted сервисам и проектам.

## Стек

- Чистый HTML5, CSS3, Vanilla JS — без фреймворков и сборщиков
- Дизайн по Material Design 3 (тёмная тема, фиолетовый акцент)
- Иконки и шрифт: Google Material Symbols + Roboto (CDN)

## Структура

```
index.html   — разметка страницы
styles.css   — стили (MD3 токены, адаптив, анимации)
app.js       — логика FAB/диалога и ripple-эффект на карточках
```

## Добавить сервис

1. Открой `index.html`
2. Скопируй любой блок `<a class="card ...">` внутри `.cards-grid`
3. Поменяй `href`, название, описание и класс цвета иконки (`card--blue`, `card--green`, `card--orange` и т.д.)
4. Сохрани файл — готово

## Запуск

Это статический сайт, никакой сборки не нужно. Достаточно открыть `index.html` в браузере или положить файлы на любой статический хостинг / веб-сервер.
